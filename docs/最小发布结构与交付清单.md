## 4. 面向普通用户的最小交付清单

如果只给单个 Windows 用户使用，最小需要：

- `game-booster-client.exe`
- 分配给该用户的唯一 `client.conf`
- 一个对应的 `client.json`
- `客户端使用说明.md`

该用户不需要拿到：

- 其他客户端配置
- 服务端私钥
- 服务端运维文档

## 6. 从零复现的最小步骤

### 客户端侧

1. 安装 `WireGuard for Windows`
2. 获取分配给本机的 `.conf`
3. 准备 `client.json`
4. 使用管理员权限运行 `connect`
5. 执行 `status`
6. `ping 10.99.0.1`
7. 联机完成后执行 `disconnect`

### 服务端侧

1. 确认 `wg-quick@wg0` 为 `active`
2. 确认 `51820/udp` 在监听
3. 确认云安全组已放行 `51820/udp`
4. 确认目标客户端 peer 已写入 `wg0.conf`
